***

Используйте в названиях правильный регистр букв:

UPPER_CASE — для глобальных констант

CamelCase — для названий классов

snake_case — для всех переменных, функций, методов, аргументов, полей

***

### Пример 

**Плохо:**
```python
constant_number = ...

class some_class():
    ...

def DO_FUNCTION():
    ...

Some_VARIABLE = ...
```
**Хорошо:**
```python
CONSTANT_NUMBER = ...

class SomeClass():
    ...

def do_function():
    ...

some_variable = ...
```
