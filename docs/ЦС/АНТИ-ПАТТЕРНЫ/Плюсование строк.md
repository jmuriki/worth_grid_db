***

Сложение строк является устаревшим решением. Для этих целей лучше использовать .format() или f-строки.

***

### Пример 

**Плохо:**
```python
name = 'Ivan'
result = 'Hello, ' + name + '! Welcome to the system.'
```
**Хорошо:**
```python
name = 'Ivan'
result = f'Hello, {name}! Welcome to the system.'
```
