
<div>
    <h1 style="margin: 0;">Излишняя вложенность</h1>
    <p style="margin: 0;">Анти-паттерн</p>
</div>

***

Увеличение уровня вложенности усложняет код, делает его менее читаемым.

***

### Пример 

**Плохо:**
```python
if status:
    if status != 'expired':
        ...
```
**Хорошо:**
```python
if status and status != 'expired':
    ...
```
